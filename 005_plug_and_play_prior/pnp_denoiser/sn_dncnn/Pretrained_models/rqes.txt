Ces poids fonctionnent pour le débruiteur de Ryu.
Le Réseau RealSN_DnCNN a été entrainé par Lucia sur ImageNet.
Pour charger le modèle, on peut utiliser la fonction load_model:

def load_model():
    path = "RealSN_DnCNN_noise2.25.pth"
    model = DnCNN(channels=1, num_of_layers=17).cuda()
    model.load_state_dict(torch.load(path))
    model.eval()
    return model
